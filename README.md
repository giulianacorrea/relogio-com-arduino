# O que é? 
Este projeto apresenta um relógio desenvolvido em C++ utilizando Arduino, criado como atividade do curso de Robótica na Educação do projeto Aranouá, uma parceria entre o IFAM e a Samsung.
O objetivo principal foi ensinar, na prática, como utilizar o Arduino, explorando conceitos básicos de programação, eletrônica e montagem de circuitos.
